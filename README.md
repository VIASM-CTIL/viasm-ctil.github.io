# Hướng dẫn sử dụng
## Tổng quan
Repository này là nơi để soạn thảo, lưu trữ, xuất bản nội dung cho website VCTIL.

- Website link: https://viasm-ctil.github.io/
- Báo vấn đề kỹ thuật: https://github.com/VIASM-CTIL/vctil-editor/issues

## Admin đăng nhập
1. Truy cập vào [trang admin](https://viasm-ctil.github.io/admin) https://viasm-ctil.github.io/admin
2. Click vào nút "Login with Github", một bảng thông báo đăng nhập sẽ hiện ra.
3. Đăng nhập với tài khoản Github. Nếu bạn có nhiều hơn 1 tài khoản, hãy chắc chắn rằng chọn tài khoản nào có quyền truy cập
vào Repository này.

## Thêm tài khoản quản lý
Tại Repository này.

1. Lựa chọn mục [Settings] -> [Manage Access] -> [Invite a collaborator]
2. Điền username của tài khoản Github của người quản lý mới. Sau đó click nút "Add". Quyền của người quản lý nên chọn từ "Write"
trở lên để đảm bảo có thể tạo và chỉnh sửa bài viết.
3. Người quản lý mới sẽ nhận được một email xác nhận. Sau khi đồng ý, việc thêm tài khoản quản lý hoàn tất.
